# UI Properties

| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |

| QlmActivationMethodTitle            | Title of the Activation Method page.                                                                                                                                                                                                                                                             |

The QlmFormatLicenseKey property must be true for this setting to have an effect.                                                                                                                                                |
